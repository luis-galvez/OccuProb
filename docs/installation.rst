Installation
============

Requirements
~~~~~~~~~~~~
- Python 3.7+
- NumPy (>=1.19)
- matplotlib (>=3.1.3)
- ASE (>=3.22.1)
- pymatgen (>=2022.0.17)

Installing via PyPI
~~~~~~~~~~~~~~~~~~~
The latest stable release can be installed from the Python Package Index (PyPI) using ``pip``:

.. code-block:: console

	pip install occuprob
